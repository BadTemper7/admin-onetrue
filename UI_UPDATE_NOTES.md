# Admin Module Layout Update

The AdminClients layout pattern was carried into these admin modules:

- Client Verification
- Admin Accounts
- Assigned Container Inventory
- Yard Area Setup

Shared layout behavior now includes:

- Summary cards inside the module header card
- Search inputs in the list toolbar
- Dropdown filters with reset controls
- Filtered record counts
- Paginated tables
- Clear action buttons
- Detail, create, edit, relocation, and decision modals where applicable
- OTLI navy, orange, slate, blue, amber, and red semantic colors

The existing API endpoints and business actions were preserved.
